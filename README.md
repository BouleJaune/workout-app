# workout-app
App workout html simple pour suivi training/calories.

/!\ vibecoded en partie /!\


## Utilisation locale
Setup environnement:
```sh
python -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
```

Lancement app:
```sh
./venv/bin/uvicorn main:app --host 127.0.0.1 --port 8080
```
